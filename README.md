# Transport Map for Posterior Sampling of Air Carbon Ablation Model Parameters
This repository contains the construction and application of a transport map fitted to posterior MCMC samples for the Air Carbon Ablation (ACA) model. The map provides a deterministic transformation from a reference distribution to the calibrated posterior, enabling efficient sampling, density evaluation, and uncertainty propagation.
The posterior distribution is obtained by calibrating ACA model parameters against molecular beam experimental data. By learning a transport map representation of the posterior, this work facilitates faster inference, reduced sampling cost, and improved exploration of parameter uncertainty.
